---
title: "Encryption"
---

Use `gpg` utility to encrypt some stuff on [[Linux]]. For example, encrypt a file with symmetric algorithm (use a single passphrase for encryption and decryption):
```shell
# encrypt
gpg -symmetric <my-file>
# decrypt
gpg -d <my-file>
```
