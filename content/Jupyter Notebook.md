Interactive notebook.

### Basics
There are three cell types - Code, Markdown and Raw.

### Shortcuts
`Ctrl + Enter` - run cell
`Shift + Enter` - run cell and move below
`Alt + Enter` - run cell and insert a cell below

`Esc + b` - create a cell
`Esc + dd` - delete a cell

`Ctrl + i` - open contextual help
